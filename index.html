Lazy Load Demonstration

Developed by: Ryan Morton

This is a demonstration of a lazy load framework. It works using IntersectionObserver and some classes. Spefically .js-lazy

Lazy-Load: Three Ways

  1. Photo assets
  2. Content flow
  3. Bonus: Time-based

Photo Assets

(Traditional Lazy Load)

This code utilizes a traditional lazy load approach of changing the src attribute on an element to the src when needed. The classic "pop in" effect is obfuscated by container elements being faded in.

Visiual

The visual animation is the same across all elements that have the .js-lazy class. This provides a clear understanding and expectation for the user.

Implementation

As long as the class .js-lazy is present on the parent element, the content will do the fade in dance. If you want to take advantage of the image lazy loading (and you should), change your src attributes to data-src.
