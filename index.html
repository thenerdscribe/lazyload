LazyLoad.js

An opinionated way to load sections of content

How it Works

Using a (polyfilled) IntersectionObserver, LazyLoad.js offloads image fetching until needed and also after a few seconds of the browser being loaded. A standard animation for lazy-loaded sections is applied.

Implementation

Wire up any element that you want to defer loading with the class of your choice. Define a second class to add for your animation. Simple as that. 

            
              const lazy = new LazyLoad(".js-lazy", "show");
              lazy.load();

Layout Agnostic

LazyLoad.js doesn't care about how your content is configured inside of the container. Slap the same class on and you'll have a good time.

Options

  • elementToWatch (Required)
  • classToAdd (Required)
  • lazyTimeout How long to wait to fetch other images
  • intersectionRatio How much of the section needs to be intersected
